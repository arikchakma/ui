<div align="center">
  <h2>🎨 ui</h2>
  <p>Collection of headless, composable UI primitives for React.</p>
</div>

### Packages

- [cromia](packages/cromia) — Unstyled, accessible color picker with composable area, sliders, inputs, and swatches
- [unit-field](packages/unit-field) — Unstyled, accessible numeric input with drag-to-scrub, arrow key stepping, and modifier key support

### Development

This is a [Turborepo](https://turbo.build/repo) monorepo using [pnpm](https://pnpm.io).

```bash
pnpm install
pnpm dev
```
